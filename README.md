# Contact Keeper

React (client-side) application to get lat&long from address.

Stack and tools: 

-  React, React Redux, Redux-thunk
-  immutable.js
-  Semantic UI

Hosted here: []()

## Install dependencies

```bash

```

### Running locally

```bash
npm run start:dev    # http://localhost:3000
```
